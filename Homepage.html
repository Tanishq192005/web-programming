History of C++

C++ was created by Bjarne Stroustrup at Bell Labs in 1979 as an enhancement to the C language. Stroustrup wanted to combine the efficiency of C with object-oriented features like classes and inheritance.

In 1983, the language was renamed C++ and introduced key features like virtual functions and function overloading. In 1998, it was officially standardized as **C++98**, and further revisions, such as **C++11**, brought new features like the **auto keyword**, **lambda expressions**, and **concurrency** support.

Key Milestones:

  • 1979: Creation of C++ (C with Classes).
  • 1983: Renamed C++ and added major features.
  • 1998: C++ standardized as **C++98**.
  • 2011: Release of **C++11**, introducing modern features like **auto** and **lambda expressions**.
  • 2020: Release of **C++20** with more powerful tools like **concepts** and **coroutines**.
